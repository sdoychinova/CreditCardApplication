1.open package manager console and run : update-database
2.Run the CreditCardApplication project in IISExpress
3.In order to create a card application go to https://localhost:44357/Application/Create. Note that the port here might be different. You can see the port when running the app in IISExperss
4.The list of already created applications can be seen on the following url  - https://localhost:44357/application/list
5.It requires .Net core 3.1
